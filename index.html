Interactive Controls

1. Selecting Objects

  • Press 1-9 → Select a shape.
  • Press 0 → Select all shapes.
  • Press Space → Toggle camera mode.

2. Scaling Objects

  • a: Decrease width | A: Increase width
  • b: Decrease height | B: Increase height
  • c: Decrease depth | C: Increase depth

3. Rotating Objects

  • i/k: Rotate around X-axis
  • o/u: Rotate around Y-axis
  • l/j: Rotate around Z-axis

4. Moving Objects

  • →: Move right | ←: Move left
  • ↑: Move up | ↓: Move down
  • ,: Move forward | .: Move backward

Choose number [1-9] and click the button below to choose an OBJ model.
